# loan-approval-ml-system

## Introduction
Ce projet vise à développer une application de Machine Learning de bout en bout
pour la prédiction de l’acceptation des demandes de prêt bancaire.
Le système automatise la prise de décision à partir des données financières
et démographiques des clients.

## Objectifs
- Analyser le jeu de données des demandes de prêt à l’aide de l’EDA
- Prétraiter les données (valeurs manquantes, déséquilibre des classes)
- Concevoir et évaluer des modèles de Machine Learning
- Suivre les expériences avec MLflow
- Déployer le modèle via une API FastAPI
- Développer une interface frontend
- Conteneuriser l’application avec Docker
